# Codecademy 90-Day Checklist (Autism-Friendly)
Use this as a calm, clear checklist. Do **one day at a time**. If a day feels hard, split it into two days.
## Daily Routine (repeat every day)
- [ ] 1) Start timer: 45–60 minutes
- [ ] 2) Do Codecademy lesson for today
- [ ] 3) Apply one tiny change to your repo
- [ ] 4) Commit + push with GitHub Desktop
- [ ] 5) Log one sentence in `LEARNING_LOG.md`

## GitHub Desktop Checklist (every change)
- [ ] Create branch (`feature/...`)
- [ ] Make small change
- [ ] Write clear commit message (`feat:`, `fix:`, `style:`)
- [ ] Push origin
- [ ] Open PR on GitHub
- [ ] Merge PR
- [ ] Pull latest `main`
