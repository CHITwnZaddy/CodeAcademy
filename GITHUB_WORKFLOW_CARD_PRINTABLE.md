# GitHub Workflow Card (Printable, One Page)

Use this for every update. Keep steps small. You are doing great.

---

## Before You Start (30 seconds)
- [ ] Drink water / adjust lights / lower sound if needed.
- [ ] Set a 25-minute timer.
- [ ] Pick exactly **one tiny task**.

---

## 8-Step Workflow Checklist

### 1) Sync `main`
- [ ] Open GitHub Desktop.
- [ ] Select branch: `main`.
- [ ] Click **Fetch origin**.
- [ ] Click **Pull origin** (if shown).

### 2) Make a new branch
- [ ] Click **Current Branch** → **New Branch**.
- [ ] Name it: `feature/short-description` (example: `feature/update-home-text`).
- [ ] Create branch from `main`.

### 3) Make one small change
- [ ] Edit file(s) in VS Code/Finder.
- [ ] Keep the change focused (one mini-goal).

### 4) Commit
- [ ] Open **Changes** tab in GitHub Desktop.
- [ ] In **Summary**, type one commit message:
  - [ ] `feat: ...` (new feature)
  - [ ] `fix: ...` (bug fix)
  - [ ] `style: ...` (visual/text only)
  - [ ] `docs: ...` (documentation)
- [ ] Click **Commit to <branch-name>**.

### 5) Push
- [ ] Click **Push origin**.

### 6) Open Pull Request
- [ ] Click **Preview Pull Request** / **Create Pull Request**.
- [ ] Confirm:
  - [ ] base = `main`
  - [ ] compare = your branch
- [ ] Click **Create pull request**.

### 7) Merge
- [ ] Click **Merge pull request**.
- [ ] Click **Confirm merge**.
- [ ] Optional: click **Delete branch**.

### 8) Clean up
- [ ] Back in GitHub Desktop, switch to `main`.
- [ ] Click **Fetch origin**.
- [ ] Click **Pull origin**.

---

## Pause Points (Overwhelm Plan)

If overwhelmed, stop and do this:
1. [ ] Close your eyes for 10 seconds.
2. [ ] Inhale 4 sec → hold 4 sec → exhale 6 sec (x3).
3. [ ] Only do the **next single click**.
4. [ ] If stuck, send screenshot and ask: “What is my next click?”
